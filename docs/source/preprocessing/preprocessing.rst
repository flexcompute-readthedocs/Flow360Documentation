Preprocessing
=============

The following section focuses on preparing a mesh for Flow360 either manually, using meshing software, or automatically, using automated meshing functionality. Flow360 solver supports the following mesh formats:

- AFLR3 (.ugrid).
- CGNS (single and multi-block).

for details of mesh formats see :ref:`here<manualMeshingFlow360Section>`.
