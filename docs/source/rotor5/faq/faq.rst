Frequently Asked Questions
==========================

.. toctree::
   :maxdepth: 2

How do I run a simulation and access the results?
-------------------------------------------------

Submitting and monitoring jobs, and donwloading the results, is all done through our web UI or python API.

How is using Rotor5 billed?
---------------------------

Rotor5, as used for designs, simulations and performance analyses, is free. We only bill the run time of the solver on our server, taking only the CPU wall clocks into account (as opposed to overhead e.g. during uploading).
